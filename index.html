SoccerPass – Demo Offline

Este é um protótipo simples em HTML para simular o fluxo do aplicativo SoccerPass de forma offline.

Funcionalidades simuladas:

  • Cadastro de times
  • Cadastro de campos
  • Agendamento de partidas
  • Simulação de pagamento e cancelamento

Abra este arquivo no navegador do seu computador ou celular para testar.
